Ultimate V11 使用說明

1. 建議先用「新遊戲」建立不同團的存檔，例如 RIIZE 元彬線、TXT 然竣線。 2. 平常玩單檔 HTML 就可以；如果要像 App，請使用 ZIP 專案部署到 GitHub Pages。 3. 記憶庫會越玩越多，建議每隔一段時間按「整理摘要」「壓縮長期記憶」。 4. 重要伏筆請釘選，例如手鍊、房卡、訊息撤回、Dispatch預告。 5. 想接 AI：到 API 設定貼 OpenAI API key，遊戲模式選 AI 自動續寫。 6. 不想接 AI：本機劇情引擎也能玩，但文字變化比較固定。 7. 存檔只在你的瀏覽器，換手機前請先「匯出全部」。

API 設定
